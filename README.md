# Mini EFootball 3D Mobile

A mobile-friendly 3D football mini-game using Babylon.js.

## Controls

### Keyboard:
- Arrow Keys: Move player
- Q/W/E/R/T/Y/U: Skill moves
- 1/2/3/4: Shoot (different types)

### Mobile:
- On-screen joystick for movement

## Features

- 3D player, opponent, and ball
- Football pitch
- Skills (stepover, chop turn, Marseille, flip-flap, rabona, double touch, no-look)
- Multiple shot types (normal, dip, finesse, stunning)
- Timer & score
- AI opponent chases ball
